# Instrutions

## User

### Login

```
user login <username>
```
### View current user

```
user view
```

## Item

### Add Item

```cmd
item create <name> <price> <selling_price> <qty>
```

### Get all item

```cmd
item all
```

### View a item

```cmd
item view <name>
```

## Order

### Place order

```cmd
order place <id> <qty>
```
